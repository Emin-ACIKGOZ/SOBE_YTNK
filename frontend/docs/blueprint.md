# **App Name**: SÖBE HR Management System

## Core Features:

- HR Login: Secure login for HR personnel to access the system.
- Job Posting Creation: Create and manage job postings with detailed descriptions and requirements.
- Job Application Tracking: Track job applications, filter candidates based on criteria, and view candidate details.
- AI Candidate Matching: Employ an AI tool that analyzes job requirements and matches the most suitable candidates based on their profiles.
- CV Upload: Drag-and-drop interface for uploading CVs in PDF format.
- PDF CV Parsing: Directly process uploaded PDF CVs to extract text. Store PDF document.
- CV Preview: Show a thumbnail and summary of uploaded PDF's

## Style Guidelines:

- Primary color: A calm blue (#5DADE2) to convey trust and professionalism.
- Background color: A very light, desaturated blue (#F0F8FF) to provide a clean and unobtrusive backdrop.
- Accent color: A soft green (#82E0AA) to highlight important actions and create a positive impression.
- Body and headline font: 'Inter' (sans-serif) for a modern and neutral look.
- Clean and intuitive layout with a focus on usability.
- Use professional and clear icons to aid navigation.
- Subtle animations to enhance user experience and provide feedback.
